##  À propos du projet

Bienvenue sur le dépôt du projet **N'oubliez Pas Les Paroles - Le Jeu** ! 
Ce site a été développé dans un but purement pédagogique, dans le cadre de mon apprentissage. Il n'a aucune vocation commerciale.

>  **Note aux ayants droit :** Ce projet est un hommage à l'émission et un exercice technique. S'il vous plaît, ne tapez pas trop fort !  Si la présence de ce projet pose le moindre problème, je suis entièrement disponible pour en discuter tranquillement et faire les modifications nécessaires.

---

##  Le But du Jeu

Tout comme dans la célèbre émission de télévision, le principe est simple : **retrouvez et devinez les paroles masquées** d'une chanson pour accumuler des points et tester vos connaissances musicales !
